Sobre

O site permite aos usuários calcular e visualizar gráficos de funções quadráticas de forma interativa. Eles inserem os coeficientes da função em um formulário e o site gera o gráfico correspondente. Além disso, possui páginas de erro personalizadas para melhorar a experiência do usuário. Com uma interface simples, é uma ferramenta conveniente para explorar funções quadráticas...

Linguagens utilizadas: HTML, CSS, JavaScript e Python

Frameworks utilizados (Python): Flask, Matplotlib e NumPy
